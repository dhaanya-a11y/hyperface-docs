## Introduction

This guide explains how to create and configure Milestone Offers in the Hyperface Dashboard.

Milestone Offers reward customers for reaching a cumulative spending or transaction-count target within a defined time period.

---

## What are Milestone Offers?

A Milestone Offer is a reward type where the customer must reach a spending or transaction-count target within a specific time period.

### Examples

- Spend 5,000 in the next 7 days and get a reward.
- Complete 5 transactions worth more than 1,000 each in the next 7 days and get a reward.

### Components

A Milestone Offer consists of:

- Offer Details
- User Base Configuration
- Milestone Eligibility
- Offer Calculation Rules
- Posting Configuration
- Display Elements

---

# Create a Milestone Offer

1. Log in to Dashboard.
2. Navigate to Portfolio Growth → Offers → Manage Offers.
3. Click Create Offer.
4. Follow the configuration journey.

---

## Step 1: Select Offer Type and Outcome

- Select **Milestone Offer**
- Click Continue
- Select Outcome Type:
  - Cashback
  - Reward Points
  - Voucher
  - Smart Tag

---

## Step 2: Configure Offer Details

### Offer Information

Configure:

- Offer Name
- Offer Description
- Start Date & Time
- End Date & Time

### Issuer and Program

Configure:

- Issuer
- Program Type
- Program Selection
- Apply To All Programs
- Progress Transfer On Variant Upgrade

---

## Step 3: Configure Offer User Base

### User Base Types

#### Static Base

- Fixed customer list
- All Accounts In Scope
- Smart Tag Filtering

#### Dynamic Config

- Opt-In
- Smart Tag Attachment

### Smart Tag Options

- Allowed
- Denied

---

## Step 4: Configure Milestone Eligibility

### Aggregation Types

#### Sum of Transaction Values

Aggregates customer spend.

#### Count of Transactions

Aggregates number of transactions.

### Threshold Configuration

Configure:

- Greater Than
- Greater Than Or Equal To
- Less Than
- Equal To

### Subscription Cycle

Options:

- Custom
- Billing Month

### Transaction Eligibility

Supported Filters:

- MCC
- MID
- Merchant Name
- Transaction Value
- Transaction Type
- POS Entry Mode

### Processing Date

Choose:

- Transaction Date
- Posting Date

---

## Step 5: Configure Offer Calculation

### Calculation Rules

Available options include:

- Flat Multiplier
- Fixed Reward
- Percentage Based
- Slab Based

### Additional Options

- Reward Point Capping
- Transaction Level Cap
- MCC Cap
- MID Cap
- Currency Cap
- Program Cap

### Reversal

Enable or disable reward reversals.

---

## Step 6: Configure Usage Limits

Optional limits:

- Daily Limit
- Weekly Limit
- Monthly Limit
- Lifetime Limit
- Cycle Count Limit

---

## Step 7: Configure Posting

### Posting Options

#### Post Immediately

Reward posts immediately after validation.

#### Scheduled Posting

Reward posts after configured delay.

### Configuration

- Enable Posting
- Posting Eligibility
- Days After
- Event
- Reward Point Expiry
- Posting Narration
- Reversal Narration

---

## Step 8: Configure Display Elements

### Home Page

Configure:

- Label
- Banner Title
- Banner Description
- Display Title
- Display Description
- Progress Style
- Display Order
- Display Colour
- Merchant Logo
- Background Illustration

### Offer Details Page

Configure:

- Terms & Conditions
- How To Redeem
- Milestone Pending Text
- Milestone Completion Text
- CTA Button Text
- Redirection Link

---

## Step 9: Review and Publish

Review:

- Offer Details
- User Base
- Milestone Eligibility
- Calculation
- Usage Limits
- Posting
- Display Elements

Click **Publish**.

The offer status becomes **Scheduled** and automatically changes to **Active** at the configured start time.

---

## Notes

- Verify all configurations before publishing.
- Active offers have limited edit options.
- Eligibility and calculation rules become locked once live.
