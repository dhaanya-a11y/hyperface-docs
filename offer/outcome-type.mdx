# Offer Type

## What is an Offer Type?

An Offer Type defines how customers become eligible to receive a reward. Each Offer Type is designed to support a specific business objective and determines the eligibility criteria, reward logic, and overall customer experience.

Hyperface supports multiple Offer Types to help issuers create reward campaigns for different business use cases.

![Offer Type Selection](/images/outcome-type-selection.png)

---

## Transactional Offer

A **Transactional Offer** rewards customers when an individual transaction satisfies the configured eligibility criteria. Each qualifying transaction is evaluated independently.

Typical use cases include cashback on fuel, dining, shopping, travel, and utility bill payments.

## Milestone Offer

A **Milestone Offer** rewards customers after successfully achieving predefined spending or transaction milestones.

Typical use cases include monthly spend campaigns and transaction count–based reward campaigns.

## Merchant Offer

A **Merchant Offer** rewards customers for making eligible purchases at participating merchants.

Typical use cases include merchant cashback campaigns and partner promotions.

## Gamified Offer

A **Gamified Offer** rewards customers through interactive campaigns and engagement-based activities.

Typical use cases include scratch cards, spin-the-wheel campaigns, and seasonal engagement programs.
