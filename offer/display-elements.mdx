# Display Elements

## What are Display Elements?

The **Display Elements** section defines how an offer is presented to customers across the Progressive Web App (PWA). These settings control the information displayed on the Home page, Offer page, and Offer Details page, ensuring a consistent customer experience across all Offer Types.

This configuration is common to **Transactional, Milestone, Merchant, and Gamified Offers**.

---

## Home Page Configuration

Configure how the offer appears on the customer's Home page.

![Step 11 - Configure Home Page](/images/m.png)

Configure the following fields:

| Field | Description | Example |
|-------|-------------|---------|
| **Enable Display** | Enables or disables the visibility of the offer in the PWA. | Enabled |
| **Labels** | Assign one or more labels to categorize the offer. | Cashback, Fuel |
| **Banner Title** | Title displayed on the offer banner. | Fuel Cashback Week |
| **Banner Description** | Short description displayed on the banner. | Earn cashback on every fuel transaction. |
| **Display Title** | Primary title displayed to customers. | 5% Fuel Cashback |
| **Display Description** | Detailed description of the offer displayed in the PWA. | Earn 5% cashback on eligible fuel transactions throughout the campaign period. |
| **Display Order** | Defines the order in which the offer appears in the PWA. | 1 |
| **Display Color** | Primary theme color used for the offer card. | #0099FF |
| **Offer/Merchant Logo** | Upload the logo displayed with the offer. | fuel-logo.png |
| **Background Illustration** | Upload the background image displayed on the offer card. | fuel-banner.png |

## Offer Details Page Configuration

Configure the detailed information displayed when customers open an offer.

![Step 11 - Configure Offer Details Page](/images/n.png)

### Section Details

| Field | Description | Example |
|-------|-------------|---------|
| **Section Title** | Title displayed for a content section. | Offer Benefits |
| **Section Content** | Rich text content describing the offer. | Get 5% cashback on eligible fuel transactions. |
| **Add Condition** | Adds another section to display additional information. | Terms & Conditions |

---

### Call-to-Action (CTA)

Configure the button displayed on the Offer Details page.

| Field | Description | Example |
|-------|-------------|---------|
| **Text on Button** | Text displayed on the CTA button. | Know More |
| **Redirection Link** | URL opened when customers click the CTA button. | https://example.com/offers/fuel |
