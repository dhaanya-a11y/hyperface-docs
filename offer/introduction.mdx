# Offer

Offers enable issuers to create and manage customer incentive programs that reward customers for performing predefined actions or achieving specific business objectives. An Offer combines eligibility rules, reward logic, usage limits, communication settings, and display configurations into a single configurable campaign.

Whether the objective is to reward a single transaction, encourage higher spending, promote partner merchants, or improve customer engagement, Hyperface provides a flexible Offer framework to support a wide range of business use cases.

Before creating an Offer, it is important to understand the core concepts involved in the Offer configuration journey. Each concept represents a specific stage of the Offer setup process and defines how the Offer behaves throughout its lifecycle.

The following sections explain each concept in detail:

- Offer Type
- Outcome Type
- Offer Scope
- Offer User Base
- Transaction Eligibility
- Offer Calculation
- Budget
- Usage Limits
- Posting
- Display Elements
- Communication
