## What is Offer User Base?

The Offer User Base defines which customers are eligible to participate in a given offer, using either a fixed list of customers or dynamic, criteria-based enrollment.

## Why is it used?

It exists so that an offer can be targeted to a specific customer segment — such as newly onboarded customers, a Smart-Tagged group, or accounts meeting particular criteria — instead of being made available to every customer in the program by default.
