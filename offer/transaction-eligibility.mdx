## What is Transaction Eligibility?

Transaction Eligibility defines the specific conditions a transaction must satisfy to qualify for a reward — for example, MCC, MID, transaction value, or transaction type.

## Why is it used?

It exists so that rewards are issued only for transactions that genuinely match the intended purchase behavior of a campaign, rather than any transaction a customer happens to make.
