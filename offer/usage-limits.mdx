# Usage Limits

## What are Usage Limits?

The **Usage Limits** section defines the maximum reward or benefit that a customer can receive within a specified period. These limits help control reward distribution, prevent excessive usage, and ensure that the offer complies with the campaign budget and business policies.

Configure one or more usage limits based on your campaign requirements.

![Step 9 - Configure Usage Limits](/images/k.png)

## Usage Limit Configuration

Configure the following fields:

| Field | Description | Example |
|-------|-------------|---------|
| **Duration** | Select the time period during which the usage limit should be applied. | Monthly |
| **Limit Type** | Select the type of limit that should be enforced for the offer. | Cashback Amount |
| **Limit Value** | Specify the maximum reward amount or usage value that a customer can receive during the selected duration. | ₹500 |

### Multiple Usage Limits

If the campaign requires more than one usage limit:

1. Click **Add More**.
2. Configure the additional usage limit.
3. Repeat the process until all required limits have been added.

This allows multiple usage limits to be applied to the same offer based on different campaign requirements.

---

## Complete the Configuration

After completing the Usage Limits configuration:

1. Review all configured usage limits.
2. Verify the duration, limit type, and limit values.
3. Click **Continue** to proceed to the **Posting** configuration.
