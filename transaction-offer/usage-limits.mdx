### Step 9: Configure Usage Limits

After configuring the **Budget**, the **Usage Limits** screen is displayed.

Use this page to define the maximum benefit that a customer can receive within a specified time period.

#### Configure Usage Limits

Provide the following information:

- **Duration** – Select the period during which the usage limit should be applied (for example, Daily, Monthly, Quarterly, or Annual).
- **Limit Type** – Select the type of limit to be enforced.
- **Limit Value** – Enter the maximum reward amount or usage value that a customer can avail during the selected period.

If multiple usage limits are required:

- Click **Add more**.
- Configure additional usage limits as required.

![Step 9 - Configure Usage Limits](/images/k.png)

After completing the configuration:

- Review the configured usage limits.
- Click **Continue** to proceed to the **Posting** configuration.
