# Apple Store Reward Points

## Overview

This guide explains how to configure the **Apple Store Reward Points** offer in the Offer Management System (OMS).

## Business objective

The **Apple Store Reward Points** offer is designed to encourage customers to use their eligible credit cards for purchases at Apple Store by rewarding qualifying transactions with **1,000 Reward Points**. Customers receive **1,000 Reward Points** on eligible Apple Store purchases of **₹15,000 or more**.
![Apple Store Reward Points](./images/apple.png)
---

## Offer type

| Field | Value |
|-------|-------|
| Offer Type | Transactional Offer |
| Offer Outcome | Reward Points |

This offer rewards customers with **1,000 Reward Points** on eligible Apple Store purchases of **₹15,000 or more**.

---

# Basic Details

Navigate to **Offer Creation → Basic Details** and configure the following fields.

## Configuration

| Field | Value | Description |
|-------|-------|-------------|
| Offer Name | Apple Store Reward Points | Name displayed for the offer. |
| Offer Description | Earn 1,000 Reward Points on eligible Apple Store purchases of ₹15,000 or more. | Short description displayed for the offer. |
| Offer Duration | 03 Jul 2026 – 24 Dec 2026 | Period during which the offer remains active. |
| Eligibility Level | Account Level | Eligibility is evaluated independently for each account. |
| Issuer | Demo-Bank | Credit card issuing bank. |
| Program Type | Credit Card | Product type for which the offer is applicable. |
| Apply to All Current and Future Programs | Disabled | Applies only to the selected program. |
| Program | IOB DEMO | Eligible credit card program. |

> **Selected Configuration:** The offer is configured at the **Account Level**, allowing eligibility to be evaluated independently for each account.

# Craft the Perfect UI: Details Page

The Details Page provides customers with complete information about the offer.

## Section Details

### How it Works

1. Visit an Apple Store.
2. Spend ₹15,000 or more using your eligible credit card.
3. The transaction is validated against the configured Merchant ID (MID).
4. Earn 1,000 Reward Points.

### Terms and Conditions

- Valid only on eligible Apple Store purchases.
- Minimum transaction amount must be ₹15,000.
- Earn 1,000 Reward Points on eligible purchases.
- Cancelled or refunded transactions are not eligible.
- Standard terms and conditions apply.

### CTA Button

| Field | Value | Description |
|-------|-------|-------------|
| Text on Button | Shop Now | CTA button displayed on the offer details page. |

> **Selected Configuration:** The details page explains the offer benefits, eligibility criteria, and Reward Points, while the **"Shop Now"** CTA encourages customers to make eligible Apple Store purchases.
