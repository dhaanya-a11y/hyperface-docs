## Create a Merchant Offer

To create a new Merchant Offer, begin from the **Manage Offers** page.

### Step 1: Create a New Merchant Offer

On the **Manage Offers** page:

- Navigate to the **Merchant Offers** tab.
- Review the existing Merchant Offers, if required.
- Click **+ Create New Offer**.
- The Merchant Offer creation journey is displayed.

<Frame>

![Step 1 - Create a Merchant Offer](/images/1.png)

</Frame>

After clicking **+ Create New Offer**, the **Offer Details** screen is displayed.

<Note>

Merchant Offers are created from the **Merchant Offers** tab under **Manage Offers**.

</Note>
