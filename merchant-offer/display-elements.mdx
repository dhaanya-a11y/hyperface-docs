## Step 4: Configure Display Elements

After configuring the **Budget**, the **Display Elements** screen is displayed.

Use this page to configure how the Merchant Offer is presented to customers in the Progressive Web Application (PWA). The Display Elements configuration consists of two sections:

- **Home Page**
- **Offer Details Page**

Configure both sections to provide a consistent and engaging customer experience.

---

### Configure the Home Page

The **Home** page allows you to define how the Merchant Offer appears on the customer's home screen.

Provide the following information:

- **Enable Display** – Enable this option to display the Merchant Offer in the customer's PWA. Disable it if the offer should remain active but should not be visible to customers.
- **Labels** – Select one or more labels to categorize the Merchant Offer.
- **Display Title** – Enter the title displayed for the Merchant Offer on the PWA.
- **Display Description** – Enter a brief description displayed below the title.
- **Display Order** – Specify the display priority of the Merchant Offer.
- **Display Color** – Select the primary color used for the Merchant Offer card.
- **Offer/Merchant Logo** – Upload the logo displayed for the Merchant Offer.
- **Background Illustration** – Upload the background image displayed on the Merchant Offer card.

- Click **Continue** to proceed to the **Offer Details Page** configuration.

<Note>

Turn **OFF** the **Enable Display** toggle if the Merchant Offer should remain active in the backend but should not be displayed to customers in the PWA.

</Note>

## Step 5: Configure the Offer Details Page

After configuring the **Home Page**, the **Offer Details Page** screen is displayed.

Use this page to configure the detailed information customers see after opening the Merchant Offer in the Progressive Web Application (PWA).

### Section Details

Use the **Section Details** editor to organize and present detailed information about the Merchant Offer.

Provide the following information:

- **Section Title** – Enter a title for the section.
- **Section Content** – Add the section content using the Rich Text Editor (RTE).
- **Add Section** – Click **Add Section** to create additional sections, such as **How It Works**, **Terms & Conditions**, or other offer-related information.
- **Reorder Sections** – Drag and drop sections to change their display order.
- **Delete Section** – Remove an existing section, if required.

### Configure the Call-to-Action (CTA)

Provide the following information:

- **Text On Button** – Enter the text displayed on the Call-to-Action (CTA) button.
- **Redirection Link** – Enter the URL to which customers are redirected after clicking the CTA button.
- **Coupon Code** – Enter the coupon code associated with the Merchant Offer, if applicable.

### Advanced Settings

Enable **Advanced Settings** to configure additional display options for the Merchant Offer, if required.
