### Step 7: Configure Display Elements

After configuring the **Posting**, the **Display Elements** screen is displayed.

Use this page to configure how the offer is presented to customers in the PWA. Configure the display elements for the **Home** page, the **Offer** page, and the **Offer Details** page to provide a consistent customer experience.

#### Configure the Home Page

The **Home** page allows you to define how the offer appears on the customer's home screen.

Provide the following information:

- **Enable Display** – Enable this option to display the offer in the customer's PWA.
- **Labels** – Select one or more labels to categorize the offer.
- **Banner Title** – Enter the title to be displayed on the offer banner.
- **Banner Description** – Enter a brief description of the offer.
- **Display Title** – Enter the title of the offer displayed to the customer.
- **Display Description** – Enter the detailed description of the offer.
- **Display Order** – Specify the display priority of the offer.
- **Display Color** – Select the primary color used for the offer card in the PWA.
- **Offer/Merchant Logo** – Upload the logo to be displayed with the offer.
- **Background Illustration** – Upload the background image displayed on the offer page.

After completing the configuration:

- Review the configured display elements.
- Click **Continue** to proceed to the **Offer Details** page configuration.

#### Configure the Offer Details Page

The **Offer Details** page allows you to configure the detailed information that customers see after opening an offer in the PWA.

**Section Details**

Use the **Section Details** editor to create and organize the content displayed on the offer details page.

Provide the following information:

- **Section Title** – Enter a title for the section.
- **Section Content** – Enter the content that describes the offer. Use the rich text editor to format the content as required.
- **Add Condition** – Click **Add Condition** to create additional sections for displaying more offer information.

**Configure Call-to-Action (CTA)**

Provide the following information:

- **Text on Button** – Enter the text to be displayed on the CTA button.
- **Redirection Link** – Enter the URL to which customers should be redirected when they click the CTA button.

**Advanced Settings**

Enable **Advanced Settings** to configure additional display options for the offer, if required.

![Step 7 - Configure Offer Details Page](/images/n.png)
![Step 7 - Configure Offer Details Settings](/images/n.1.png)

After completing the configuration:

- Review the configured Offer Details page settings.
- Click **Continue** to proceed to the **Communication** configuration.
