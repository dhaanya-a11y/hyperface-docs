## Step 1: Select Offer Type

After creating a new offer, the **Offer Type** screen is displayed.

Use this page to select the type of offer you want to create.

Select **Milestone Offers** under the **Transaction Data** category.

The available offer categories include:

- All Benefits
- Transaction Data
- EMI Creation
- Card Events
- Display Only

Under **Transaction Data**, you can create:

- Transaction Offers
- Milestone Offers
- Gamified Offers

<Frame>

![Step 1 - Select Offer Type](/images/0.2.png)

</Frame>

After completing the configuration:

- Select **Milestone Offers**.
- Click **Continue** to proceed to the **Outcome Type** configuration.

<Note>

Milestone Offers reward customers after they successfully complete one or more configured milestones.

</Note>
