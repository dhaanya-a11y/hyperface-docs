## Create a Milestone Offer

To create a new Milestone Offer, begin from the **Manage Offers** page.

### Step 1: Create a New Milestone Offer

On the **Manage Offers** page:

- Navigate to the **CVP Offers** tab.
- Click **+ Create New Offer**.
- Select **Single Offer**.
- The Milestone Offer creation journey is displayed.

<Frame>

![Step 1 - Create a New Milestone Offer](/images/0.1.png)

</Frame>

After selecting **Single Offer**, the **Offer Type** screen is displayed.

<Note>

Single Offers allow you to configure one offer at a time.

</Note>
