## Step 2: Select Outcome Type

After selecting the offer type, the **Outcome Type** screen is displayed.

Use this page to define the reward customers receive after completing the configured milestone.

Select one of the available outcome types.

The available outcome types include:

- Cashback
- Reward
- Voucher
- Smart Tag
- Partner Points

Some outcome types may appear as **Coming Soon** and are not currently available for selection.

<Frame>

![Step 2 - Select Outcome Type](/images/0.3.png)

</Frame>

After completing the configuration:

- Select the required outcome type.
- Click **Continue** to proceed to the **Offer Details** configuration.
