## Step 4: Configure Offer User Base

After configuring the **Offer Details**, the **Offer User Base** screen is displayed.

Use this page to define which customers are eligible to participate in the Milestone Offer.

### Select the User Base Type

Choose the appropriate user base type based on your campaign requirements.

The following user base options are available:

| User Base Type | Description |
|----------------|-------------|
| **Static Base** | Uses a fixed list of eligible customers throughout the offer period. |
| **Dynamic Config** | Continuously updates eligible customers based on the configured eligibility criteria until the offer becomes active. |

### Configure Eligibility

After selecting the user base type, configure the eligibility criteria used to identify eligible customers.

The available eligibility options include:

- Account Enrollment
- Opt-In
- Smart Tag Attachment
- Cycle-Wise Enrollment

<Frame>

![Step 4 - Configure Offer User Base](/images/0.5.png)

</Frame>

After completing the configuration:

- Configure the required eligibility criteria.
- Click **Estimate User Base** to calculate the estimated number of eligible customers.
- Review the estimated user count.
- Click **Continue** to proceed to the **Milestone Eligibility** configuration.

<Note>

The configured eligibility determines which customers are allowed to participate in the Milestone Offer.

</Note>
