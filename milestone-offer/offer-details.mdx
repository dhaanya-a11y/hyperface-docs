## Step 3: Configure Offer Details

After selecting the outcome type, the **Offer Details** screen is displayed.

Use this page to configure the basic information required for the Milestone Offer and define where the offer will be applicable.

### Basic Offer Details

Provide the following information:

- **Offer Name** – Enter a unique name to identify the Milestone Offer.
- **Offer Description** – Enter a brief description of the offer.
- **Offer Duration** – Select the start and end date during which the offer remains active.
- **Eligibility Level** – Select how milestone eligibility should be evaluated.
  - **Account Level** – Evaluates eligibility independently for each account.
  - **Customer Level** – Evaluates eligibility across all accounts belonging to the customer.

### Offer Applicability

Configure where the Milestone Offer should be available.

Provide the following information:

- **Issuer** – Select the issuing bank.
- **Program Type** – Select the applicable program type.
- **Apply to all current and future programs** – Enable this option to automatically apply the Milestone Offer across all existing and future programs.
- **Program** – Select one or more applicable programs.
- **Enable progress transfer on variant upgrade** – Enable this option to retain milestone progress when an eligible customer upgrades to another supported card variant.

<Frame>

![Step 3 - Configure Offer Applicability](/images/0.4.1.png)

</Frame>

After completing the configuration:

- Review the configured offer details.
- Click **Continue** to proceed to the **Offer User Base** configuration.

<Note>

If **Apply to all current and future programs** is enabled, the Milestone Offer is automatically associated with all existing and future programs for the selected Program Type.

</Note>
