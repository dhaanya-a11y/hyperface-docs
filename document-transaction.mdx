## What is a Transactional Offer?

<Info>
A **Transactional Offer** rewards customers for completing a **single qualifying transaction** that satisfies the configured eligibility criteria. Each qualifying transaction is evaluated independently, and eligible customers receive the configured reward based on the selected outcome type.
</Info>

---

## Create a Transactional Offer

To create a new Transactional Offer, begin from the **Manage Offers** page.

### Step 1: Create a New Offer

On the **Manage Offers** page:

- Click **+ Create New Offer** located in the upper-right corner.
- From the available options, select **Single Offer**.
- The offer creation journey is displayed.

![Step 1 - Create a Transactional Offer](/images/a.png)

---

### Step 2: Select the Offer Type

After selecting **Single Offer**, the **Offer Type** screen is displayed.

On this screen:

- Browse the available offer types.
- Select **Transaction Offers**.
- Click **Continue**.
- The **Outcome Type** screen is displayed.

![Step 2 - Select the Offer Type](/images/c.png)

---

### Step 3: Select the Outcome Type

After clicking **Continue**, the **Outcome Type** screen is displayed.

The Outcome Type determines the reward that customers receive after completing a qualifying transaction.

On this screen:

- Browse the available outcome types.
- Select the required outcome type.
- Available options include:
  - Cashback
  - Reward
  - Voucher
  - Smart Tag
  - Accelerated Reward
  - Partner Points
- Click **Continue**.
- The **Offer Scope** screen is displayed.

![Step 3 - Select the Outcome Type](/images/d.png)

---

---

### Step 4: Configure the Offer Scope

After selecting the **Outcome Type**, the **Offer Scope** screen is displayed.

Use this screen to configure the basic details of the Transactional Offer and define where the offer will be applicable.

#### Configure the Basic Offer Details

Enter the following information:

- **Offer Name** – Enter a name to identify the offer.
- **Offer Description** – Enter a brief description of the offer.
- **Offer Duration** – Select the start and end date and time during which the offer will remain active.
- **Eligibility Level** – Select how the offer eligibility should be evaluated:
  - **Account Level** – Evaluates eligibility independently for each account.
  - **Customer Level** – Evaluates eligibility across all accounts belonging to the customer.

![Step 4 - Configure Basic Offer Details](/images/e.png)

#### Configure Offer Applicability

Continue configuring where the offer should be applicable.

Configure the following details:

- **Issuer** – Select the issuer for which the offer will be applicable.
- **Program Type** – Select the applicable program type.
- **Apply to all current and future programs** – Enable this option to make the offer available across all existing and future programs.
- **Program** – If the above option is disabled, select one or more applicable programs from the **Program** dropdown.

After reviewing the configured details, click **Continue** to proceed to the **Set-Up Offer** configuration.

![Step 4 - Configure Offer Applicability](/images/f.png)
