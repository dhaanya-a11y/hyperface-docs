## Create a Transactional Offer

To create a new Transactional Offer, begin from the **Manage Offers** page.

### Step 1: Click Create New Offer

On the **Manage Offers** page:

- Click **+ Create New Offer** in the upper-right corner.
- A list of available offer creation options is displayed.

![Step 1 - Create New Offer](/images/a.png)

### Step 2: Select Single Offer

From the available options:

- Select **Single Offer**.
- This starts the offer creation journey.
- The **Offer Type** selection screen is displayed.

![Step 2 - Select Single Offer](/images/b.png)
