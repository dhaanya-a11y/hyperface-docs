## Create a Transactional Offer

To create a new Transactional Offer, begin from the **Manage Offers** page.

### Step 1: Click Create New Offer

On the **Manage Offers** page, click **+ Create New Offer** located in the upper-right corner. This opens a menu with the available offer creation options.

![Step 1 - Create New Offer](/images/transactional-offers/a.png)

### Step 2: Select Single Offer

From the available options, select **Single Offer** to create an individual offer. This starts the offer creation journey and opens the Offer Type selection screen.

![Step 2 - Select Single Offer](/images/transactional-offers/b.png)
