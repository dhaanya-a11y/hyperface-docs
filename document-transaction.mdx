## Create a Transactional Offer

To create a new Transactional Offer, begin from the **Manage Offers** page.

### Step 1: Click Create New Offer

On the **Manage Offers** page:

- Click **+ Create New Offer** in the upper-right corner.
- A list of available offer creation options is displayed.

![Step 1 - Create New Offer](/images/a.png)

### Step 2: Select Single Offer

From the available options:

- Select **Single Offer**.
- This starts the offer creation journey.
- The **Offer Type** selection screen is displayed.

![Step 2 - Select Single Offer](/images/b.png)

### Step 3: Select Transaction Offers

On the **Offer Type** screen:

- Select **Transaction Offers** from the list of available offer types.
- Click **Continue** to proceed to the **Outcome Type** screen.

![Step 3 - Select Transaction Offers](/images/c.png)

### Step 4: Select the Outcome Type

On the **Outcome Type** screen:

- Select the outcome type that the customer should receive after completing a qualifying transaction.
- Available options include:
  - Cashback
  - Reward
  - Voucher
  - Smart Tag
  - Accelerated Reward
  - Partner Points
- After selecting the required outcome type, click **Continue**.

![Step 4 - Select Outcome Type](/images/d.png)
