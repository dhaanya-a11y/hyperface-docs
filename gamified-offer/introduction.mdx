# Gamified Offer

## What are Gamified Offers?

A Gamified Offer is a reward type where customers unlock multiple rewards by crossing a series of predefined thresholds — or **levels** — within a specified period.

Unlike a Milestone Offer, which tracks progress toward a single target, a Gamified Offer defines **multiple thresholds**, each mapped to its **own reward**. As a customer's cumulative activity grows, they move up through the levels, unlocking a new offer every time a threshold is crossed.

Once all configurations are completed and reviewed, the offer is published and becomes available to eligible customers from the configured start date.

This guide explains how to create and configure Gamified Offers in the Hyperface Dashboard.

Gamified Offers reward customers progressively as they cross multiple spending or transaction-count thresholds within a defined time period, turning reward earning into a level-by-level experience.

---

### Key Characteristics

Gamified Offers track customer progress across multiple thresholds within a defined subscription cycle, unlocking a distinct reward at each level.

Key characteristics include:

* Multiple thresholds (levels) are configured within a single offer.
* Each threshold has its own independent reward configuration.
* Progress is accumulated across multiple eligible transactions, same as Milestone Offers.
* Customers can unlock more than one level within the same cycle if cumulative activity crosses multiple thresholds.
* Levels can be based on total spend or transaction count.
* Subscription cycles determine the duration over which activity is measured and levels can reset.
* Transaction-level eligibility rules determine which transactions contribute toward level progress.
* Customer-facing UI displays progress as a level ladder or progress bar, showing the next unlockable reward.

Gamified Offers are commonly used for tiered spend campaigns, engagement and retention programs, seasonal reward drives, and any campaign designed to motivate incremental spend behaviour.

### Gamified Offers vs. Milestone Offers

| | Gamified Offer | Milestone Offer |
|---|---|---|
| **Thresholds** | Multiple thresholds (levels), each with its own reward | Single threshold |
| **Rewards** | One reward per level, unlocked progressively | One reward, unlocked once |
| **Customer experience** | Level-by-level, game-like progression | Single target completion |
| **Typical use case** | Tiered spend campaigns, engagement drives | Simple spend/transaction-count targets |

For details on Milestone Offers, see [Milestone Offers](/milestone-offer/introduction).

### Examples

- Level 1: Spend ₹5,000 → Earn ₹200 cashback. Level 2: Spend ₹10,000 → Earn ₹500 cashback. Level 3: Spend ₹20,000 → Earn ₹1,200 cashback — all within the same billing month.
- Level 1: Complete 3 transactions → Earn 100 reward points. Level 2: Complete 7 transactions → Earn 300 reward points. Level 3: Complete 12 transactions → Earn a Voucher.

### Components

A Gamified Offer consists of:

- Offer Details
- User Base Configuration
- Gamification Levels (Threshold Ladder)
- Offer Calculation Rules (per level)
- Posting Configuration
- Display Elements
