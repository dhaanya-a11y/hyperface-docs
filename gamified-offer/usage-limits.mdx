## Step 6: Configure Usage Limits

Optional limits:

- Daily Limit
- Weekly Limit
- Monthly Limit
- Lifetime Limit
- Cycle Count Limit
- Level Unlock Limit (maximum number of levels a customer can unlock per cycle)

### Usage Limit Examples

- Cycle Count Limit = 3 (the full level ladder can be completed up to 3 times)
- Level Unlock Limit = 3 (all configured levels can be unlocked once per cycle)
- Lifetime Limit = 25 rewards during the offer period

Combined limits can be used to manage reward exposure and campaign costs.

### Usage Limit Best Practices

Usage limits help control campaign costs and prevent excessive reward earning, which is especially important in Gamified Offers where a single customer can unlock multiple rewards within one cycle.

Before publishing:

- Review expected reward exposure across the full level ladder.
- Configure limits based on campaign objectives.
- Validate cycle count restrictions.
- Ensure limits align with budget allocations.

If no limits are configured, customers may continue earning rewards at every level whenever threshold conditions are satisfied.
