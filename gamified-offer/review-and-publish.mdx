## Step 9: Review and Publish

### Pre-Publish Validation Checklist

Before publishing a Gamified Offer, verify:

* Offer Details are accurate.
* User Base configuration is correct.
* All Gamification Levels — thresholds, in ascending order — have been validated.
* Reward calculations have been tested for every level.
* Usage Limits align with campaign objectives across the full ladder.
* Posting Configuration is complete.
* Display content, including the level ladder, has been reviewed and approved.

Publishing should only be performed after all configurations have been reviewed and validated.

Review all configurations before publishing:

- Offer Details
- User Base
- Gamification Levels
- Calculation Rules (per level)
- Usage Limits
- Posting Configuration
- Display Elements

Click **Publish**.

The offer status becomes **Scheduled** and automatically changes to **Active** at the configured start time.
