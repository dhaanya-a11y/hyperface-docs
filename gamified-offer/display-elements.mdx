## Step 8: Configure Display Elements

Display Elements control how the offer is presented to customers across the Hyperface platform, including how the level ladder and current progress are visualized.

### Home Page Configuration

Configure:

- Label
- Banner Title
- Banner Description
- Display Title
- Display Description
- Progress Style (Level Ladder, Progress Bar, or Badge Stepper)
- Display Order
- Display Colour
- Merchant Logo
- Background Illustration

### Offer Details Page Configuration

Configure:

- Terms & Conditions
- How To Redeem
- Level Pending Text (per level, shown while the customer is progressing toward that level)
- Level Unlocked Text (per level, shown once that level's reward is earned)
- CTA Button Text
- Redirection Link

<Note>
Since a Gamified Offer contains multiple levels, Level Pending Text and Level Unlocked Text should be configured for each level so customers always see accurate progress messaging for their current stage.
</Note>
